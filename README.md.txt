# 🎬 Netflix Movies Data Analysis

## 📌 Project Overview

This project explores Netflix movie data using Python to uncover trends in genres, popularity, voting patterns, and movie releases. Through Exploratory Data Analysis (EDA), valuable insights were extracted and visualized using charts and graphs.

### 🎯 Business Questions Answered

1. What is the most frequent movie genre?
2. Which vote category has the highest count?
3. Which movie has the highest popularity and what is its genre?
4. Which movie has the lowest popularity and what is its genre?
5. Which year has the highest number of movie releases?

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Analysis & Visualizations

## 📈 Key Findings

### Most Frequent Genre
- Drama is the most common genre on Netflix.

### Highest Popularity Movie
- Spider-Man: No Way Home
- Genres: Action, Adventure, Science Fiction
- Popularity Score: 5083.954

### Lowest Popularity Movies
- The United States vs. Billie Holiday
- Threads

### Release Trend
- The highest number of movies were released between 2010 and 2020.

### Vote Analysis
- Most movies belong to Average and Popular vote categories.


## 🚀 Conclusion

This project demonstrates how Exploratory Data Analysis can reveal meaningful insights from movie datasets. The analysis identified dominant genres, popularity trends, voting behavior, and release patterns, helping understand audience preferences and content distribution on Netflix.
